Цель

гарантировать, что у класса будет только один единственный экземпляр и к нему будет предоставлена глобальная точка доступа

для чего
Для создания единственного экземпляра определенного класса

ПРимер
- необходим только один экземпляр конкретного класса, который доступн для всех клиентов
- единственный экземпляр должын наследоваться путем порождения подклассов, при этом, клиенты
имеют возможность работать с классом наследником без модификации своего кода
